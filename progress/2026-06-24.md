### 踩坑记录

1. **花括号转义**: prompt 中 JSON 示例 `{}` 被 `str.format()` 当成占位符 → 全部 `{{}}` 转义
2. **模型名**: vllm 注册全路径 `/ky200t/models/Qwen2.5-32B-Instruct`，传 HF 短名返回 404
3. **KV cache 不足**: max_model_len=32768 需 8GB KV cache → 降到 16384，GPU 0 单卡够
4. **max_tokens 超限**: compress 默认 max_tokens=16384，prompt 8193 tokens 挤爆 → 降为 4096
5. **GPU 空挂**: 多张卡有进程残留（0% util），`fuser -k` 清理后恢复正常
6. **SSH 断连**: nohup 进程不受影响，但 tmux 窗口会丢
